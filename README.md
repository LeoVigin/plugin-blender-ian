This project is a Blender addon that displays a live MP4 preview window with subtitles for simple and fast tutorials

## Installation & Start
- Download the folder as a ZIP or clone it using
``` php
git clone https://github.com/LeoVigin/plugin-blender-ian.git
```
- Open ```pyside6_download.py``` and run it in Blender's Scripting workspace (you can remove it afterward).
- Either install it directly from your disk via Blenderâ€™s Add-ons menu, or run the script from its path.

##  How To Use The Addon
- Open the video player: <br>
```Ctrl + Shift + P```
- Select a tool from the interface.
- Toggle subtitles: <br>
```Ctrl + S```

### Now, Enjoy!
